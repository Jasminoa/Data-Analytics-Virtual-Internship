This project involves cleaning a raw campaign dataset (Campaign_Data_Q3.csv) to prepare it for analysis and visualization. The goal was to ensure data consistency, remove quality issues, and structure the dataset for reporting in tools like Excel or Power BI.

The following issues were identified in the raw dataset:
- Inconsistent formatting across columns
- No missing values detected
- No duplicate records found

The following cleaning steps were performed:

1. Standardized column formatting for consistency
2. Unified text and data formats across all fields
3. Verified dataset integrity by checking for duplicates
4. Confirmed there were no missing values

Tools used:
- Microsoft Excel

After cleaning:
- No missing values
- No duplicate records
- Fully standardized dataset ready for analysis
