# Data Analytics Virtual Internship


## Project Overview
This project simulates a real-world data analytics workflow using a campaign dataset (Q3). It includes data cleaning, exploratory analysis, and preparation for visualization and reporting.

## Key Work Completed
- Cleaned and standardized raw campaign dataset
- Checked and confirmed data quality (no missing values or duplicates)
- Structured dataset for analysis and reporting
- Documented full data cleaning process

## Repository Structure
